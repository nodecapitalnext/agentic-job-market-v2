# AgenticJobMarket - ERC-8183 Implementation

Full-featured AI Agent Job Market on Arc Testnet.

## Setup

```bash
cp .env.example .env
# Fill in your PRIVATE_KEY in .env (never commit it!)
forge install
forge build
forge test
```
